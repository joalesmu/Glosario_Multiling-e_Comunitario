# 🌿 Glosario Multimedia Comunitario (con opción de uso Sin Internet)

> **Versión:** 0.9 beta 
> **Licencia:** MIT (Código Abierto)  
> **México | 2026**

Un sistema integral diseñado para la preservación, documentación y revitalización de lenguas originarias en contextos sin acceso a internet. Este proyecto prioriza la soberanía de los datos comunitarios y la dignidad de todas las variantes lingüísticas.

## 🚀 Características Principales

* **📱 Registro Sin Conexión:** Aplicación móvil (AppSheet) que funciona totalmente sin internet para trabajo de campo.
* **🧠 Gestión Centralizada:** Base de datos en Google Sheets con scripts de automatización.
* **🌐 Visor Web Universal:** Interfaz HTML ligera para consultar el glosario en cualquier dispositivo.
* **🤝 Fusión Inteligente:** Herramienta de importación compatible con estándares de Macrolenguas para evitar duplicados y fragmentación.
* **☁️ Sincronización:** Carga automática de audios y fotos a la nube al detectar conexión.

## 📂 Estructura del Repositorio

Este repositorio organiza los recursos necesarios para implementar el sistema:

* **`/src`:** Código fuente del sistema (`.gs`) y Visor Web (`.html`).
* **`/docs`:** Manuales de usuario, guías de instalación y plantillas de datos.

## 🛠️ Instalación

Para implementar tu propia copia de este sistema:

1.  **Copia la App:** Utiliza el enlace público de AppSheet proporcionado en la documentación.
2.  **Configura el Cerebro:** Copia el contenido de `src/CODIGO_SISTEMA.txt` en tu Hoja de Cálculo de Google.
3.  **Despliega:** Publica el script como Aplicación Web.
4.  **Consulta:** Usa GitHub Pages o el archivo local `index.html`.
